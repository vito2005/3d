# hemi.xyz 3D components demo

This repository is a small **Next.js demo app** showcasing several **3D components for `hemi.xyz`**.

All 3D components in `app/components/*-3d.tsx` were originally made for **Framer** and then adapted to run inside this Next.js project.

Demo link - `https://3d-one-sandy.vercel.app/`

## Development

```bash
npm install
npm run dev
```

Then open `http://localhost:3000`.
